# task1-cssgrid
## Сверстайте адаптивную страницу по образцу с фотографии используя css-grid
На мобильной версии информация в секции hero должна располагаться в одну колонку. На планшетной и компьютерной версии в две колонки.
---
Карточки должны отображаться в одну колонку на мобильной, две колонки на планшетной, три колонки на компьютере.
---
Используем функции auto-fit и minmax. Минимальная ширина для колонки в hero-секции - 350 пикселей, для карточек - 280 пикселей.
## Начало работы
1. Создайте ветку dev
2. Сделайте основную вёрстку (цвета, шрифты, отступы). Сделайте коммит "#1 Добавлены базовые стили".
3. Добавьте стили grid для секции hero. Сделайте коммит.
4. Добавьте стили grid для секции с карточками. Сделайте коммит.
5. Создайте запрос на вливание (create pull request). В комментарии к запросу допишите "Задание выполнено" и Фамилии и Имена выполнивших.
6. Сделайте слияние веток.
